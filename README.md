# 💱 Currency Calculator

A simple and user-friendly Currency Calculator web application that allows users to convert one currency into another using selectable country currencies and flags.

---

## 📌 Features

- Convert currency between different countries  
- Swap currencies with one click  
- Country flags update automatically  
- Clean and responsive UI  
- Fast and lightweight  
- Works on mobile and desktop  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6)  
- Flags API  
- Remix Icons
